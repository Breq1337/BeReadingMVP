O QUE FALTA (depende do usuario)
================================

1. SUPABASE ANON KEY
   - Ir no dashboard Supabase > Settings > API
   - Copiar a "anon public" key
   - Colar no .env.local no lugar de PLACEHOLDER_NEED_ANON_KEY

2. RODAR MIGRACAO SQL
   - Abrir SQL Editor no Supabase
   - Colar conteudo do supabase-migration.sql
   - Executar

3. DEPLOY VERCEL
   - Configurar variaveis de ambiente no Vercel
   - NEXT_PUBLIC_SUPABASE_URL e NEXT_PUBLIC_SUPABASE_ANON_KEY

4. CONTEUDO REAL
   - Capas de livros reais (Open Library ja fornece)
   - Missoes reais por livro
   - Dados da escola
